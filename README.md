# KnackAPI
Methods to help work with the Knack API.

## Use in NodeJS:

Install the package
```
$ npm install knack-api-helper --save
```

Use it in your js code
```javascript
const KnackAPI = require('knack-api-helper');

const knackAPI = new KnackAPI({
    auth: 'object-based',
    apiKey: 'YOUR-API-KEY'
});
//More to come here
```
